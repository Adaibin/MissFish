时光未老，理想还在。

昨天学到了两个template language的有意思用法。 一个是set，一个是super()。

set是在模板中对变量赋值(不存在则创建)，这个变量可被导出到其子模板使用，也可导入到父模板使用 : {% set path = [('/user_index', '用户管理'), ('/user_edit', '编辑用户')] %}。

super() 调用父级块的内容来渲染，这会返回父级块的结果: {% block body %}{{ super() }}{% endblock %}。
